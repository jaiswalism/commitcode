How to Configure CommitCode

1

Create an empty repository

Create a brand new repository specifically for CommitCode (e.g. leetcode-solutions). Do NOT point it at an existing personal repo.

2

Generate a Fine-grained PAT

Create a Personal Access Token so the extension can safely write to your new repository.

A.Go to Settings > Credentials > Fine-grained personal access token and click Generate new token.

B.Under Repository access, select "Only select repositories" and choose your new repo.

C.Under Permissions, give Contents access to Read and write. Click Generate.

3

Connect and Verify

Paste the token and your repo (format: username/repo-name) into the fields, and click Test Connection to verify.

Security Note: Your token is stored locally on your device via Chrome's local storage. It is never transmitted anywhere except directly to GitHub's API.
